A tool for annotating images with lines.


* Free software: Allen Institute Software License

* Documentation: https://sarcomereannotation.readthedocs.io.

Installation
------------

`> git clone :repo_url:`

`> pip install -e .`

Usage
-----

`> line_annot`

or

`> line_annot -ip :path/to/image:`

Features
--------

* load an image
* draw lines on image
* edit lines that have been added
